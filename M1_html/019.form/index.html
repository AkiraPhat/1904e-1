Thẻ biểu mẫu form trong html

   *** Thẻ form có 3 thuộc tính quan trọng
    name : tên của form
    method : mang 2 giá trị post hoặc get ( khi học php )
    action : địa chỉ mà form sẽ gửi dữ liệu đến
    *** thẻ input
    có type="text" => văn bản
    thẻ input có thuộc tính name ( tên của input )
    value="" ( giá trị của ô input )
    placeholder="" ( hướng dẫn cho người dùng hiểu ô input làm gì nó sẽ xuất hiện khi mà chưa nhập
    giá trị vào ô input đó )
    *** thẻ label là thẻ nhãn
